# StoreCraft

Marketing site for **StoreCraft** — an independent UK studio building Shopify
stores and local-business websites for small businesses.

🔗 **Live:** https://vaelariss.github.io/portfolio/

## Structure

| File | Purpose |
|------|---------|
| `index.html` | Main landing page |
| `styles.css` / `script.js` | Shared styles + behaviour for the landing page |
| `bloom-ember.html` | Demo build — handmade candle brand (Shopify concept) |
| `aurum-jewels.html` | Demo build — jewellery brand (Shopify concept) |
| `fade-district.html` | Demo build — barbershop (brochure concept) |
| `privacy.html` / `terms.html` | Legal pages |
| `404.html` | Custom not-found page |
| `og-cover.png`, `favicon.svg`, `apple-touch-icon.png`, `favicon-32.png` | Branding / social preview assets |
| `robots.txt`, `sitemap.xml` | Crawl / indexing |

The three demo pages are **concept builds** created to showcase design and
layout. "Parallax" (linked from the Work section) is a live Shopify store.

## Tech

Hand-written static HTML/CSS/JS — no build step, no dependencies. Hosted on
GitHub Pages. The contact form posts to Formspree. Fonts via Google Fonts.
